# AI-SPEECH-APP

## Description

This web application is designed to help users improve their speech skills through AI-driven feedback. Users can sign up or log in to access personalised speech prompts. The application records their speech, analyses it using an AI model, and provides detailed feedback and statistics to guide their improvement. Whether youâ€™re practicing for a presentation or just looking to enhance your speaking abilities, this tool offers a tailored experience to boost your confidence and proficiency.

## Features

- Login/Sign Up
- Generated promopts to speak
- Record Speech
- Speech Analysed by AI model
- Given Feedback/Stats for improvement
